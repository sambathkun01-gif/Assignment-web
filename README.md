3-User Features
- Register & login with password hashing (bcrypt)
- Browse all movies with poster, genre, and average rating
- Search movies by title or description
- Filter movies by genre
- View movie detail page with all reviews
- Rate movies 1–5 stars and write a comment
- Edit or delete your own reviews
- Personal dashboard showing all your reviews
- Edit profile info and change password
- Upload profile photo
- Dark / Light mode toggle (saved in localStorage)
4-Admin Features
- Admin dashboard with stats (movies, users, reviews)
- Add, edit, delete movies with poster image upload
- Add, edit, delete users
- View and delete any review
- Protected routes (non-admins redirected)
